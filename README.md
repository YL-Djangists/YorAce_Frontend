# YorAce_Frontend

> [!WARNING]
> Предпологается что все действия будт происходить из корня папки

## Клонирование проекта

```cmd
git clone https://github.com/YL-Djangists/YorAce_Frontend
cd ./YorAce_Frontend/
```

## Установка зависимостей

```cmd
cd ./yorace/
npm install
```

## Запуск проекта

```cmd
cd ./yorace/
npm start
```
